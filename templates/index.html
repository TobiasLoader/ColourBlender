about the project.

Hi, this is Toby Loader, known on the web as TobiasCodes. A bit of background on myself, I'm a student at the University of Oxford reading Maths and CS and love writing code and coming up with fun little projects to work on.

Having done lots of frontend dev over the years I've often needed to find colour schemes for my projects. But I never found an online tool that suited my needs whilst still being easy and fun to use. So I've built myself this little helper and hope it will be of use to some people.

If you would like to watch a video of me building the web app you can check out my youtube channel TobiasCodes :)

show colour css.

Below is the css for the colour you've selected. This is a linear interpolation (lerp) of the left and right colours, at the percentage 'split'%. So 0% would represent all of the left colour and none of the right, whilst 100% would be none of the left and all of the right.

Use the toggle in the top right of the window to switch between hex and rgb colour encodings.
